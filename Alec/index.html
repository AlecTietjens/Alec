    Badge Distribution Tracker

    Summer 2015

    I developed a solution where a massive 200,000+ badge redistribution with hundreds to thousands of shipments can be tracked. The solution uses data collected from the database/systems used for onsite tracking as well as information gathered (through APIs) from external carriers, i.e., FedEx, UPS, and DHL. Many technologies were used.

    Web-based Ticketing Tool

    Spring 2015

    A tool was developed for a client who needed a more efficient way for customers to create request tickets. I developed a web-based (ASP.NET) tool to allow for creation of tickets as well as a management piece to be used for tracking and workflow.

    Software Deployment Tool

    Fall 2014

    A medium-sized company wanted a cheap and efficient way to manage their computers, so I developed a PowerShell solution to deploy software, uninstall software, and keep logs for managed computers through PowerShell.
